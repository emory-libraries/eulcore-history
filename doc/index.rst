EULcore |release|
=================

EULlocal is a collection of reusable Python_ components from `Emory
University Libraries`_. The library contains both released and
unreleased beta components. Except where noted otherwise, components
documented here are released and ready for production use.

Modules that were formerly part of :mod:`eulcore` have been split out
into :mod:`eulxml`, :mod:`eulfedora`, :mod:`eulexistdb`, and
:mod:`eulcommon`.

.. _Python: http://www.python.org/
.. _Emory University Libraries: http://web.library.emory.edu/
